Política de Privacidade

Última atualização: 18 de novembro de 2025

1. Introdução

Este aplicativo é um sistema interno de gestão de processos contábeis via WhatsApp. Sua privacidade é importante para nós.

2. Informações Coletadas

Coletamos apenas:

  • Número de telefone do WhatsApp (para autenticação)
  • Comandos enviados via mensagem (para processar solicitações)
  • Logs de acesso (para segurança e auditoria)

3. Uso das Informações

As informações são usadas exclusivamente para:

  • Autenticar usuários autorizados
  • Processar e responder comandos
  • Gerar relatórios e análises solicitadas
  • Manter logs de auditoria interna

4. Compartilhamento de Dados

Não compartilhamos suas informações com terceiros. Os dados permanecem internos ao sistema.

5. Segurança

Implementamos medidas de segurança para proteger suas informações:

  • Autenticação por número de telefone autorizado
  • Validação de assinaturas webhook (HMAC)
  • Logs criptografados
  • Acesso restrito apenas a gestores autorizados

6. Retenção de Dados

Logs são mantidos por até 90 dias para fins de auditoria e depois são automaticamente excluídos.

7. Seus Direitos

Você tem direito a:

  • Solicitar acesso aos seus dados
  • Solicitar correção de dados incorretos
  • Solicitar exclusão dos seus dados
  • Revogar acesso ao sistema

8. Cookies

Este aplicativo não utiliza cookies, pois funciona exclusivamente via WhatsApp.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Mudanças serão notificadas via WhatsApp aos usuários autorizados.

10. Contato

Para questões sobre esta política de privacidade, entre em contato através do WhatsApp: +55 62 99273-1445

11. Conformidade LGPD

Este aplicativo está em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).
